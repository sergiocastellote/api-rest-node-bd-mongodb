#Configure

- Clone project
- Install mongodb
- Execute 'npm install'
- Execute 'npm start'

#Api

- server: http://localhost:3000/
- get all: http://localhost:3000/books/
- get one: http://localhost:3000/books/:id
- post: http://localhost:3000/books/
- put: http://localhost:3000/books/:id
- delete: http://localhost:3000/books/:id
